# Vue 3 + Vite + Vuetify + Vuex Base Template

This repository serves as a clean, ready-to-use boilerplate for starting new front-end applications using:
- **Vue 3** (Composition API supported)
- **Vite** (Fast building and HMR)
- **Vuetify 3** (Material Design Components)
- **Vue Router 4** (Routing)
- **Vuex 4** (State Management)

## Getting Started

1. Clone or copy this repository to your new project folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Workflow scripts

- `npm run dev` - Starts Vite development server
- `npm run build` - Compiles and bundles for production
- `npm run lint` - Lints `.js` and `.vue` files with ESLint
